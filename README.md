DBMS Project 

-------------------------------------------------
📊 Project Overview
-------------------------------------------------
This project is a Database Management System (DBMS) application 
designed to manage and analyze car resale value data.
It demonstrates concepts of database design, normalization, 
SQL queries, and integration with Python/Streamlit for visualization.

-------------------------------------------------
🛠 Tech Stack
-------------------------------------------------
- Database: MySQL / PostgreSQL
- Backend: Python (with SQL connector)
- Frontend: Streamlit (for interactive dashboards)
- Tools: ER Diagrams, Normalization, SQL Joins, Aggregate Functions

⚙️ Setup Instructions
-------------------------------------------------
1. Clone the repository
   git clone https://github.com/anishambad/DBMS-Project.git
   cd DBMS-Project

2. Install dependencies
   pip install -r requirements.txt

3. Import database schema
   Open MySQL/PostgreSQL
   Run:
     source schema.sql;

4. Run the application
   streamlit run app.py

-------------------------------------------------
📊 Features
-------------------------------------------------
- ER diagram showcasing database design
- Normalized schema (up to 3NF)
- SQL queries for CRUD operations
- Analytical queries (joins, group by, aggregate functions)
- Streamlit dashboard for visualization
